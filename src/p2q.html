To create qr code

1. `import qrcode`: This line imports the `qrcode` library, which provides functionality for generating QR codes.

2. `qr = qrcode.QRCode(version=1, error_correction=qrcode.constants.ERROR_CORRECT_H, box_size=10, border=4)`: Here, we create an instance of the `QRCode` class from the `qrcode` library. We specify the version, error correction level, box size, and border size for the QR code.

3. `data = "Hello, World!"`: This line defines the data you want to encode in the QR code. In this example, the data is set to "Hello, World!", but you can change it to any text or data you want.

4. `qr.add_data(data)`: The `add_data` method is used to add the data to the QR code instance.

5. `qr.make(fit=True)`: This line generates the QR code based on the provided data. The `fit=True` parameter allows the QR code to automatically adjust its size to fit the data.

6. `qr_image = qr.make_image(fill_color="black", back_color="white")`: Here, we create an image representation of the QR code using the `make_image` method. The `fill_color` parameter sets the color of the QR code modules (in this case, black), and the `back_color` parameter sets the background color (in this case, white).

7. `qr_image.save("qr_code.png")`: Finally, we save the generated QR code image as "qr_code.png". You can change the filename and extension to suit your needs.

This code generates a QR code containing the text "Hello, World!" and saves it as an image file. You can then use the image file for various purposes, such as sharing or printing the QR code.

     

import qrcode

# Create a QR code instance

qr = qrcode.QRCode(version=1, error_correction=qrcode.constants.ERROR_CORRECT_H, box_size=10, border=4)

# Define the data to encode in the QR code

data = "Hello, World!"

# Add data to the QR code instance

qr.add_data(data)

qr.make(fit=True)

# Create an image from the QR code instance

qr_image = qr.make_image(fill_color="black", back_color="white")

# Save the image

qr_image.save("qr_code.png")
