To create a pie chart in Python

1. `import matplotlib.pyplot as plt`: This line imports the `matplotlib.pyplot` module, which provides functions for creating various types of plots, including pie charts.

2. `labels = ['Category A', 'Category B', 'Category C', 'Category D']`: Here, we define the labels for the categories in the pie chart. Each label corresponds to a specific value.

3. `values = [30, 50, 20, 10]`: This line defines the values associated with each category. The values represent the proportions or sizes of each category in the pie chart.

4. `plt.pie(values, labels=labels, autopct='%1.1f%%')`: The `plt.pie()` function is used to create the pie chart. It takes the `values` as input and automatically calculates the corresponding proportions. The `labels` parameter is used to specify the labels for each category, and the `autopct` parameter formats the percentage values displayed on the chart to have one decimal place.

5. `plt.title('Pie Chart Example')`: This line adds a title to the pie chart. You can modify the string argument to set a different title for your chart.

6. `plt.show()`: Finally, `plt.show()` is called to display the pie chart. This will open a new window containing the chart.

When you run the code, you'll see a pie chart displayed with the specified labels, values, and percentages. Each slice of the pie represents a category, and its size is determined by the corresponding value.

Feel free to modify the code to suit your specific needs, such as changing the labels, values, colors, or adding additional formatting options to the pie chart. The `matplotlib.pyplot` module provides a wide range of customization options for creating visually appealing and informative pie charts.

import matplotlib.pyplot as plt

# Define the labels and values for the pie chart

labels = ['Category A', 'Category B', 'Category C', 'Category D']

values = [30, 50, 20, 10]

# Create the pie chart

plt.pie(values, labels=labels, autopct='%1.1f%%')

# Add a title

plt.title('Pie Chart Example')

# Display the chart

plt.show()
