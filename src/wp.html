import pywhatkit

phone_number = "1234567890"

message = "Hello, this is a message from Python."

pywhatkit.sendwhatmsg(phone_number, message, 10, 30)

This code will send the message "Hello, this is a message from Python." to the phone number 1234567890 at 10:30 AM.

Here is a breakdown of the code:

  • The import pywhatkit line imports the pywhatkit library, which is used to send messages in WhatsApp via Python.
  • The phone_number variable stores the phone number of the recipient.
  • The message variable stores the message that you want to send.
  • The pywhatkit.sendwhatmsg() function sends the message to the recipient. The first argument is the phone number, the second argument is the message, the third argument is the hour, and the fourth argument is the minutes.
