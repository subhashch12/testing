Send an email using Python:

Python

import smtplib

def send_email(to_email, subject, message):

    """Sends an email to the given address."""

    server = smtplib.SMTP("smtp.gmail.com", 587)

    server.starttls()

    server.login("your_email_address", "your_password")

    server.sendmail("your_email_address", to_email, f"Subject: {subject}\n\n{message}")

    server.quit()

if __name__ == "__main__":

    subject = "This is a test email"

    message = "This is the body of the email."

    send_email(to_email, subject, message)

The code then checks if the code is being run as a script (i.e., if it is being called from the command line). If it is, the code calls the send_email() function with the recipient's email address, the subject of the email, and the body of the email as input.
