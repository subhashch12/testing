Convert a color image to Black/ white image

The code demonstrates how to convert a color image to grayscale using the OpenCV library in Python. Here's a breakdown of the code:

1. The first line imports the OpenCV library, which is used for computer vision tasks including image processing.
2. The second line loads a color image from a file named 'your_image.jpg' using the `cv2.imread()` function. The image is stored in the `color_image` variable as a NumPy array.
3. The third line converts the color image to grayscale using the `cv2.cvtColor()` function. It takes two arguments: the `color_image` array and `cv2.COLOR_BGR2GRAY`, which specifies the conversion from BGR (Blue, Green, Red) color space to grayscale.
4. The resulting grayscale image is stored in the `gray_image` variable as a NumPy array.
5. The last line saves the grayscale image to a file named 'black_white_image.jpg' using the `cv2.imwrite()` function. It takes two arguments: the output file name and the image array (`gray_image`).

To use this code, you need to have the OpenCV library installed and have an image file named 'your_image.jpg' in the same directory as your Python script or Jupyter Notebook. After running the code, a new grayscale image named 'black_white_image.jpg' will be saved in the same directory.

     

import cv2

# Load the color image

color_image = cv2.imread('your_image.jpg')

# Convert the color image to grayscale

gray_image = cv2.cvtColor(color_image, cv2.COLOR_BGR2GRAY)

# Save the grayscale image

cv2.imwrite('black_white_image.jpg', gray_image)
