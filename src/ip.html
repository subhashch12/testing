To retrieve the IP address using Python

In this example, we first use socket.gethostname() to retrieve the hostname of the machine. Then, we pass the hostname to socket.gethostbyname() to get the corresponding IP address. Finally, we print the hostname and IP address.

When you run this code, it will display the hostname and IP address of the machine on which the script is executed.

import socket

# Get the hostname

hostname = socket.gethostname()

# Get the IP address

ip_address = socket.gethostbyname(hostname)

print(f"Hostname: {hostname}")

print(f"IP Address: {ip_address}")
